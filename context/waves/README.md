# Waves

This directory is the execution ledger for larger project goals.

The old numbered wave directories are preserved as project history. New work
uses the active-wave pattern in `PHASES.md`: pick the active wave, execute the
next unchecked pulse, update the pulse and wave card, then run the declared
validation.

Use the local skills:

- `/r-wave status`: find the active wave and next pulse.
- `/r-fork NN`: materialize pulse `NN` into a single context file.
- `/r-pulse NN`: execute pulse `NN` against the codebase.
- `/r-review pulse NN`: run role review and write panel findings.
