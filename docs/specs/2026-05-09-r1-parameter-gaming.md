## Claims
1. U.2 established that parameter tuning within the DIA's permissible range (ufactor 0.1-5%, acounty 1-10, T 100-1000) produces maximum 0.3 D-seat variation nationally.
2. An adversarial agent choosing the most Republican-favorable combination (ufactor=5%, acounty=10, T=1000) vs. the most Democratic-favorable (ufactor=0.1%, acounty=1, T=100) produces a national partisan shift of 0.3 D-seats — not a strategically significant gerrymander.
3. The DIA's parameter pre-registration requirement (parameters announced before Census data is released, fixed in the SHA-256 seed formula) eliminates any post-data parameter optimization.
4. Conclusion: parameter gaming is ineffective (0.3 seats max) and detectable (audit chain shows any parameter change from the pre-registered value).
