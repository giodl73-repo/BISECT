# Goal: Method-Produced RPLAN Package Expansion

**Status:** Active - first method package generated
**Date:** 2026-05-11
**Owner:** BISECT / RPLAN integration
**Depends on:** `docs/specs/2026-05-11-rplan-golden-package-corpus-goal.md`

## Goal

Promote the tiny golden-package fixed point into a small set of
method-produced packages: artifacts generated by real CLI or crate workflows
rather than hand-sized documentation fixtures.

This is not a benchmark milestone. The purpose is publication-grade fixture
depth: command transcripts, package manifests, certificates, method transcripts,
and verifier agreement for non-toy runs that are still small enough to review
and keep in the repository.

## Package Tiers

| Tier | Meaning | Storage |
|---|---|---|
| Tiny golden | 3x3 verifier-facing examples | `docs/examples/rplan-golden-packages/` |
| Method-produced fixture | Small generated package with transcript and stable seed | `docs/examples/rplan-method-packages/` |
| Benchmark package | Larger empirical run with external data and timing notes | Future artifact store or release bundle |

This goal targets the method-produced fixture tier.
