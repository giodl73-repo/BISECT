## Claims
1. The Iowa model (legislative staff drawing maps, no political input) is the most algorithmic existing redistricting process and provides the closest precedent for bisect adoption.
2. Three integration modes are possible: (a) algorithm as sole map-drawer (pure algorithmic), (b) algorithm as first draft (commission refines), (c) algorithm as audit tool (commission draws, algorithm scores). Mode (b) is the most politically feasible.
3. Under mode (b), the commission's adjustment authority should be limited to documented VRA compliance and community of interest concerns, with each adjustment requiring a written Gingles justification. This is already required in the D-track model statute.
4. Process design: the commission receives the bisect plan on day 1, holds public comment for 30 days, may make adjustments only within defined parameters, and must re-run bisect to certify the adjusted plan satisfies compactness within 5% of the original.
