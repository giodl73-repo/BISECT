# Provisional Packet: The Input-Exclusion Defense

This packet is the current human-review front door for The Districting Review
Vol. 1. The issue is in **provisional public preview**, not locked publication.

## Approved Strategy

The editor approved the qualitative preview path. Vol. 1 explains how Bisect's
default workflow excludes partisan data, what that proves, and how to describe
the defense without turning it into a universal legal claim.

See [Editor Approval](decisions/editor-approval-qualitative-preview.md).

## Reader Promise

Bisect's core claim is not trust. It is input exclusion: the baseline run does
not ingest partisan data. This volume tests what that does and does not prove
for practitioners.
