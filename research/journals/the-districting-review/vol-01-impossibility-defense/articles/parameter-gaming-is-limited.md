# Parameter Gaming Is Limited

An adversarial test is useful only if it stays an adversarial test.

The point of the parameter-gaming paper is not that the algorithm cannot be
influenced at all. The point is that the most partisan allowed parameter
combination still produces a bounded effect. That is a strong practical answer
to the claim that "the software can be tuned to anything."

The public language should stay careful. Stress-testing the parameter space is
evidence about boundedness, not a statement that every possible downstream use
is harmless or legally neutral.
