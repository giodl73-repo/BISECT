## Editorial Question

Can Bisect be described as a baseline mapmaker whose default workflow excludes
partisan data, without pretending that this alone settles legal neutrality?

## Audition Recommendation

Use a six-piece preview:

1. Trust problem and why architecture matters.
2. Input exclusion as the core argument.
3. Seed sensitivity as bounded variation.
4. Parameter gaming as a stress test.
5. Pre-registration and manifest binding.
6. Court-facing language that stays inside the evidence.

Do not make the issue about universal impossibility. Make it about what the
baseline workflow can honestly claim.
