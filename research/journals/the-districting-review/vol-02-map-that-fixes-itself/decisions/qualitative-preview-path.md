# Qualitative Preview Path

## Decision

Use the **qualitative preview path** for Vol. 2.

The public value is in showing output and verification without pretending that
the output answers every legal or statistical question.

## What This Allows

- Use [B11] to show the map output.
- Use [G1] to contextualize ensemble compactness.
- Use [A4] and [R4] to show verification.
- Use [D1] as a caveated VRA example.
- Use [C2] and [B15] to show stability across cycles.

## What This Blocks

- Universal legal results.
- Raw percentile claims without ESS correction.
- Turning national map outputs into claims of proof beyond the run.

## Rationale

The issue should help a practitioner understand what the tool makes and what
those outputs do not yet prove.
