# VRA Mode Needs Caveats

VRA mode belongs in the issue because practitioners will ask about it. But it
should be presented as a bounded use case, not as a universal legal result.

The article should explain what changes when the algorithm is run under VRA
constraints and why those outputs need separate caveats. The point is to show
how the tool behaves under a different specification, not to declare that one
set of outputs is legally mandatory everywhere.
