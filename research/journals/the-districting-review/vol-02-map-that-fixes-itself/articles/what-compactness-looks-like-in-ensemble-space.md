# What Compactness Looks Like in Ensemble Space

Ensemble position gives the reader context, but it has to be stated carefully.
The compactness story is about where the plan sits relative to the sampled plan
space, not about a mystical perfect rank.

Because the effective sample size is still a constraint, the public version
should use ESS-corrected language and avoid pretending a preliminary percentile
is final.
