# The Map That Fixes Itself

This volume is the output-facing practitioner issue. It shows what Bisect makes
from public data, how reproducible those outputs are, and which headline numbers
still need source-chain discipline before they can be treated as public copy.

## Current Posture

The issue is in provisional public preview. It should be useful to
commissioners and staff without turning concrete output into an unsupported
legal or statistical conclusion.

Do not draft this issue as adoption mechanics. It is about visible output,
replication, and caveats.

## Gates

- BOUNDARY: do not convert map output into a universal legal or constitutional verdict.
- SCALE: keep ensemble position and percentile language ESS-corrected.
- DATUM: every headline number needs a run name, version, and caveat.
- COMMONS: the NC result should be readable without burying the caveat stack.
