# Qualitative Preview Path

## Decision

Use the **qualitative preview path** for Vol. 3.

The public value is in usable language, not in pretending the adoption question
is fully settled.

## What This Allows

- Use [P0] for framing.
- Use [P1] for a federal statute template.
- Use [P2] for ballot initiative language.
- Use [P3] for commission operating rules.
- Use [P4] for court-order templates.
- Use [P5] for cost and procurement drafting.

## What This Blocks

- Universal mandate language.
- Legal advice framing.
- Procurement claims without assumptions.
- Court orders without remedy context.

## Rationale

Practitioners need draftable text more than another abstract argument.
