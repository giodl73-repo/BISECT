# Ballot Initiative Language

The ballot path needs state-specific text. The issue should show how citizens
could add an algorithmic baseline requirement without rewriting the whole
commission structure.

The point is not that every state should use the same words. The point is that
the draft can be adapted by states that already use initiative mechanisms.
