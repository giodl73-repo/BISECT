# Implementation Costs and Procurement

Administrators need to know what the thing costs. The issue should give a plain
cost story and a usable procurement outline without pretending the estimate is
free of assumptions.

That makes the skeptic-facing case concrete: the problem is not whether the
algorithm can be implemented. It is how a state would budget for and buy it.
