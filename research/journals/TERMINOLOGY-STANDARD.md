# Terminology Standard

Use conservative public language until an issue is locked.

## Project Terms

| Prefer | Avoid in public locked copy | Note |
|--------|-----------------------------|------|
| Bisect | redistricting AI | Bisect is a deterministic algorithmic redistricting system. |
| algorithmic baseline | mandatory algorithmic redistricting | "Mandatory" is advocacy/legal language; use only in adoption issues. |
| candidate federal rule | the federal rule | The Districting Integrity Act remains a proposal unless enacted. |
| evidence standard | proof | Courts decide what counts as proof. |
| reproducible audit chain | tamper-proof democracy | Strong metaphors can stay in editorials only after COVENANT review. |
| geographically constrained outcome | geography is destiny | The phrase works as a title, not as a literal claim. |

## Claim Verbs

| Claim maturity | Use |
|----------------|-----|
| seed | proposes, tests, asks, candidates, would evaluate |
| audition | compares, screens, selects, flags |
| provisional | finds in this corpus, estimates, reports, argues |
| review | passes/fails source-chain review, requires caveat |
| locked | shows, demonstrates, recommends, with stated scope |

## Public Caveat

Locked issues should include this posture in plain language:

> Bisect issues separate results from recommendations. A result describes what a
> run, dataset, paper, or review found. A recommendation states what a legislature,
> court, commission, or advocate might do with that evidence.
