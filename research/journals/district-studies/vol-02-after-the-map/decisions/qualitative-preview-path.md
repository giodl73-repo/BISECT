# Qualitative Preview Path

## Decision

Use the **qualitative preview path** for District Studies Vol. 2.

Vol. 2 should explain which downstream outcome claims are promising and how the
evidence differs by claim type. It should not wait for exact O.1/O.4 source
artifacts before becoming useful, but it also should not publish exact O.1/O.4
numbers in preview form.

## What This Allows

- Use [O4] to explain constituent distance as a concrete access outcome.
- Use [O1] to explain competitiveness as the first political outcome.
- Use [O2] and [O3] as examples of claims that need more causal caution.
- Use [O5] to explain why composite indexes are useful but risky.
- Publish preview summaries without exact outcome values.

## What This Blocks

- Exact O.1 competitive-district percentages.
- Exact O.4 travel-time savings.
- O.2 causal turnout claims.
- O.3 causal polarization claims.
- O.5 legal-testimony language.
- RQI threshold language.

## Rationale

The public value of Vol. 2 is the ordering of evidence. A reader should leave
understanding that not all outcome claims have the same evidentiary strength.
That lesson does not require exact values.
