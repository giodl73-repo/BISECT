# Three State Types

T.5 works best here as a qualitative case-study frame. Instead of treating
geography as a single universal answer, the frame presents three state
behaviors:

1. A narrow improvement window, where a proportionality constraint may move a
   result toward proportionality under specific settings.
2. Constraint-induced worsening, where adding partisan information can make the
   proportionality result worse.
3. Granularity and overcorrection, where small delegations or coarse partition
   choices create jumps rather than smooth adjustments.

The public example is simple: two states can have the same statewide vote split
but different voter geography. If one party's voters are densely concentrated
while the other party's voters are spread across many areas, compact districts
will face a different feasible plan space.

That is why this issue does not present geography as a slogan. Geography is a
constraint field. In some states, it may leave little room for a proportionality
adjustment. In others, an added constraint may interact with geography in
unexpected ways. In smaller delegations, one seat can be a large step, so the
result changes in jumps rather than smooth fractions.

## Public Example

The three types can be described without naming exact values:

- some states leave little room for a proportionality adjustment;
- some states can move in the wrong direction under a constraint because the
  constraint interacts with geography;
- some states have seat counts that move in jumps rather than smooth steps.

That is enough for the first issue. The table belongs later, after provenance
closure.
