# What This Issue Does Not Show

This closing piece is the issue's guardrail.

The issue does not show that geography determines every partisan
outcome, that every non-proportional result is natural, that any enacted map is
unconstitutional, or that a federal rule follows from this issue alone.

The result-bearing measurement slot is also deferred: efficiency-gap code
convention has been corrected, but public numeric claims need post-fix output
provenance before they can be used.

The guardrail keeps the issue diagnostic. It tells the reader what has been
learned, what is still gated, and what claims belong in later journals.

The main lesson is methodological. A public reader should not jump from
"partisan asymmetry exists" to a single explanation. The first issue asks
whether geography explains some of the observed outcome. Later issues can ask
what changes after the map, what a commission should do, how evidence should be
calibrated, and whether the process can be gamed.

## Public Example

If an enacted plan has a partisan skew, this issue does not by itself say why.
The skew might be partly geographic, partly intentional, partly produced by
legal constraints, or partly produced by measurement choices. Vol. 1 teaches
the geography question so later issues can separate those explanations more
carefully.
