# The Ensemble Median Case

A middle-of-the-distribution ensemble result can be informative rather than
disappointing.

In the G.1 ensemble run, the Bisect plan lands near the middle of the
compactness distribution for North Carolina. This draft treats that as a
preliminary ensemble placement, not as a certified percentile. The point is
that a deterministic compactness baseline and a random valid-plan ensemble can
converge when geography tightly constrains the feasible plan space.

The intuition is straightforward. If a baseline plan sits far outside the
comparison distribution, the result demands explanation. If it sits near the
middle, the result does not prove correctness, but it does suggest that the
baseline is not obviously an outlier under that comparison. In a geography-first
issue, that is the useful observation.

The draft uses ESS-safe language:

- "in this ensemble run";
- "near the middle";
- "preliminary point estimate";
- "multi-chain validation remains a review gate."

It does not use legal-certificate language, and it does not claim that the
North Carolina pattern generalizes to every state.

## Public Example

The useful reader takeaway is not "the plan scored exactly X." It is that a
deterministic baseline and a comparison ensemble can point to the same broad
region of the plan space. When that happens, the result is evidence that the
baseline is not obviously an outlier under the comparison method.
