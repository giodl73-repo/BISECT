# Scope-Down Path

## Decision

Use the **scope-down path** for District Studies Vol. 1.

Vol. 1 should be a clean diagnostic opening issue. It should explain why voter
geography can constrain partisan outcomes, but it should not wait for every
quantitative source-chain gate before becoming useful.

## What This Allows

- Use B.12 as the lead formal source for the geography/proportionality frame.
- Use G.1 as a cautious ensemble-position example.
- Use B.12 state contrasts qualitatively.
- Publish article summaries that teach the frame without exact table values.
- Keep L.1/C.5 as a deferred measurement problem.

## What This Blocks

- Exact B.12 table values.
- Claims about the exact B.12 paper-run METIS parameter vector.
- Exact G.1 percentile claims.
- Result-bearing L.1/C.5 efficiency-gap conclusions.
- Legal or adoption conclusions.

## Rationale

The first public issue has a reader job: introduce the diagnostic frame. That
job does not require exact quantitative table claims. The exact values matter,
but they are better handled after source-chain closure or in a later methods
issue.

## Follow-Up

1. Expand article summaries with examples and source notes.
2. Keep source requests open for later quantitative work.
3. Mark the issue as audition-ready, not locked.
4. Re-run article-summary review after the expanded summaries.
