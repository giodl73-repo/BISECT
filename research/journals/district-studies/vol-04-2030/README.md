# 2030

This volume is the forward-planning issue for District Studies. It asks what
must be ready before the 2031 redistricting window opens: the projection source
chain, the data infrastructure, the state-by-state resolution rule, and the
algorithmic implications of seat changes.

## Current Posture

This issue is in provisional public preview. The public version should present
the 2030 cycle as a planning problem with projection uncertainty, not as a
settled forecast.

Do not lock the issue until projection sources, state-boundary language, and
infrastructure claims have been reviewed together.

## Gates

- SCALE: projection uncertainty must be visible, including boundary states.
- DATUM: all seat-change and infrastructure claims need source-vintage notes.
- CONTOUR: pre-registration and build assumptions need explicit provenance.
- COMMONS: the issue must feel urgent without sounding like a countdown gimmick.
