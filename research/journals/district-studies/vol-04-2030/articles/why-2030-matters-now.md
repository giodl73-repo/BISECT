# Why 2030 Matters Now

The 2030 cycle matters now because the work needed for redistricting starts
before the census data arrives. If states wait for the 2031 release window to
begin building adjacency graphs, validating seed formulas, and checking their
data pipelines, they will be late.

This is not a countdown piece. It is a sequencing piece. The issue is about the
order of work, not panic.

The basic lesson is simple: the 2031 map process will be compressed. Any system
that expects to stand up new infrastructure after the data is published will be
starting from behind.
