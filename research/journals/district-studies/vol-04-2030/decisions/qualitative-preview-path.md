# Qualitative Preview Path

## Decision

Use the **qualitative preview path** for District Studies Vol. 4.

Vol. 4 is strongest when it turns projections into planning guidance without
overpromising precision. The issue should explain how the 2030 cycle changes
the work that must be done now.

## What This Allows

- Use [Q0] to explain the timeline and why planning begins before 2031.
- Use [Q1] to explain projected seat boundaries and the six boundary states.
- Use [Q2] to explain how Sun Belt growth affects algorithm structure.
- Use [Q3] to explain the data build required for 2030.
- Use [Q4] to explain reapportionment projections and stability implications.

## What This Blocks

- Final apportionment language.
- Any implied legal conclusion about deadlines or compliance.
- Structural recommendations without validation labels.
- Exact seat-count language without source-vintage context.

## Rationale

The preview is valuable because it tells operators what to prepare. The issue
does not need to wait for 2031 to do that work, but it does need to keep the
numbers framed as projections.
