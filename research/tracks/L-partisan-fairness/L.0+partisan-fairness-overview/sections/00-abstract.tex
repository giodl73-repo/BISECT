Partisan gerrymandering has survived federal constitutional challenge since \textit{Rucho v.\ Common Cause}
(2019), which held that federal courts lack judicially manageable standards to adjudicate such claims.
State courts remain open, but practitioners litigating under state constitutional ``free and equal elections''
clauses must navigate six partially overlapping partisan fairness metrics---Efficiency Gap (EG),
Mean-Median Difference (MM), Partisan Bias, Declination, Responsiveness (via the seats-votes curve),
and Gallagher Proportionality---each measuring a geometrically distinct property of a redistricting plan.
This paper provides the first unified framework integrating all six metrics, with a taxonomy
distinguishing descriptive metrics (EG, MM, Bias, Declination) from normative standards
(Proportionality, Responsiveness), and a decision table mapping legal theory to metric choice
for expert witnesses.

We introduce \textsc{bisect} algorithmic maps as a natural reference distribution anchored near
the partisan-neutral centroid across all six metrics. Using VEST 2020 presidential precinct returns
for North Carolina ($k=14$), Wisconsin ($k=8$), Texas ($k=38$), and Florida ($k=28$), we show
that \textsc{bisect} maps achieve lower partisan metric magnitudes in NC/WI/TX
single-seed comparisons$^\dagger$ relative to enacted 2022 congressional maps on every
descriptive metric and approach competitive neutrality on both normative standards;
multi-seed validation is required to characterize typical algorithmic behavior.
The reference distribution concept is central to post-\textit{Rucho} state court litigation:
when a court asks ``how different is this map from what a neutral process would produce?'',
the \textsc{bisect} distribution provides an empirically grounded, non-partisan answer.
