\section{Verifier Traceability}

The examples above are L1 package fixtures. They are regenerated by
\texttt{rcount-io} examples, verified by \texttt{rcount-cli}, and summarized by
\texttt{rcount-audit} transcripts.

\begin{quote}
\footnotesize
\begin{verbatim}
cargo run -p rcount-io --example canvass_correction_package
cargo run -p rcount-io --example mail_batch_added_package
cargo run -p rcount-io --example missing_batch_package

cargo run -p rcount-cli -- verify \
  docs/examples/rcount-golden-packages/canvass-correction
cargo run -p rcount-cli -- verify \
  docs/examples/rcount-golden-packages/mail-batch-added
cargo run -p rcount-cli -- verify \
  docs/examples/rcount-golden-packages/missing-batch
\end{verbatim}
\end{quote}

\begin{center}
\small
\begin{tabularx}{\textwidth}{lXX}
\toprule
Fixture & Expected result & Checks that carry the paper claim \\
\midrule
\texttt{canvass-correction} & pass & \texttt{contest\_selection\_sum}, \texttt{jurisdiction\_contest\_total}, \texttt{canvass\_correction\_event}, \texttt{source\_hash\_match}. \\
\texttt{mail-batch-added} & pass & \texttt{accepted\_ballots}, \texttt{batch\_summary\_total}, \texttt{jurisdiction\_contest\_total}, \texttt{source\_hash\_match}. \\
\texttt{missing-batch} & fail & \texttt{batch\_summary\_total} fails because a summary references \texttt{batch:P-001:late-mail} with no manifest. \\
\bottomrule
\end{tabularx}
\end{center}

A transcript excerpt for the negative fixture is intentionally specific:
\begin{quote}
\footnotesize
\begin{verbatim}
{
  "status": "fail",
  "checks": [
    {
      "equation_id": "batch_summary_total",
      "status": "fail",
      "error": "summary ... references missing batch id: batch:P-001:late-mail"
    },
    {"equation_id": "jurisdiction_contest_total", "status": "pass"},
    {"equation_id": "source_hash_match", "status": "pass"}
  ]
}
\end{verbatim}
\end{quote}

That pattern is the point of canvass arithmetic. A package can preserve source
hashes and jurisdiction totals while still failing because one batch-level
piece of evidence is absent. Public readers get the failure layer, not a vague
accusation.
