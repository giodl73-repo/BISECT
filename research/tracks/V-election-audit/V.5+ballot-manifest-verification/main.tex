\begin{abstract}
Public vote totals are not meaningful unless the counted inputs are accounted
for. This paper defines the first RCOUNT ballot-manifest verification layer:
batch manifest records must reconcile accepted, counted, and rejected ballot
counts, and each batch-level summary must reference an existing manifest row.
Using the \texttt{mail-batch-added} and \texttt{missing-batch} fixtures, we
show how RCOUNT can distinguish a legitimate late mail batch from a canvassed
summary that points at absent batch evidence.
\end{abstract}
