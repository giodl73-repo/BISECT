\section{Method}

\subsection{Permutation Test}

For a query plan with test statistic $q$ and an ensemble of $N$ plans with
statistics $x_1, \ldots, x_N$ (all computed on the same graph using the same metric):

\begin{enumerate}
  \item Compute the raw empirical p-value:
    $\hat{p} = |\{i : x_i \leq q\}| / N$.
  \item Retrieve $\ess$ from G.4's ESS table (Table~1 in G.4, corrected version).
  \item Apply the ESS correction: the ESS-corrected p-value is the posterior
    median of $\mathrm{Beta}(\hat{p} \cdot \ess + 1,\ (1-\hat{p}) \cdot \ess + 1)$,
    computed as described in S.2.
\end{enumerate}

The implementation in \texttt{permutation\_test\_lower\_tail} (Algorithm~\ref{alg:perm})
handles both one-sided (lower-tail) and the symmetry-based two-sided tests.

\begin{algorithm}
\caption{ESS-corrected permutation test}
\label{alg:perm}
\begin{algorithmic}[1]
\Require $q$: query plan statistic; $\mathbf{x} = [x_1,\ldots,x_N]$: ensemble statistics; $\mathrm{ESS}$
\State $k \leftarrow |\{i : x_i \leq q\}|$
\State $\hat{p} \leftarrow k/N$
\State $a \leftarrow \hat{p} \cdot \mathrm{ESS} + 1$; $b \leftarrow (1-\hat{p}) \cdot \mathrm{ESS} + 1$
\State $p_{\mathrm{corr}} \leftarrow$ posterior median of $\mathrm{Beta}(a, b)$ $\approx (a-1/3)/(a+b-2/3)$
\State BDS $\leftarrow I_{0.05}(a, b)$ \Comment{Bayesian Detection Score at $\tau=0.05$}
\State \Return $(k, N, \hat{p}, \mathrm{ESS}, p_{\mathrm{corr}}, \mathrm{BDS})$
\end{algorithmic}
\end{algorithm}

\subsection{ESS Values}

We use the G.4 corrected ESS table (2026-05-09 revision, Table~1).
For the 10{,}000-step ensembles in the primary analysis:
NC $\ess = 695$, WI $= 811$, GA $= 638$, PA $= 582$, TX $= 417$.

For the 1{,}000-step ensembles available from G.1:
NC $\ess \approx 70$, WI $\approx 81$, GA $\approx 64$, PA $\approx 58$, TX $\approx 42$.
