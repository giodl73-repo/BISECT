\subsection{Threshold Equivalences}

State statutes that mandate $S \leq c$ are equivalent to PP thresholds:
\[
  S \leq c \iff \text{PP} \geq \frac{1}{c^2}.
\]
Key equivalences:
\begin{itemize}
  \item $S \leq 1.5$ $\Leftrightarrow$ PP $\geq 0.44$
  \item $S \leq 1.8$ $\Leftrightarrow$ PP $\geq 0.31$
  \item $S \leq 2.0$ $\Leftrightarrow$ PP $\geq 0.25$
  \item $S \leq 2.5$ $\Leftrightarrow$ PP $\geq 0.16$
  \item $S \leq 3.0$ $\Leftrightarrow$ PP $\geq 0.11$
\end{itemize}

\subsection{Practical Conversion in bisect}

The CLI converts PP to S via:
\begin{verbatim}
bisect label-analyze <label> --year 2020 --types schwartzberg
\end{verbatim}
Internally, the implementation computes PP and then applies $S = 1/\sqrt{\text{PP}}$.
The equivalence $|S - 1/\sqrt{\text{PP}}| < 10^{-6}$ is verified in the L0 test suite.
