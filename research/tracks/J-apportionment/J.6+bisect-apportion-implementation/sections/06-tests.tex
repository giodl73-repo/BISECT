\label{sec:tests}

The \textsc{bisect-apportion} test suite currently comprises 99 tests;
24 tests specifically cover apportionment methods and paradox checking
(\texttt{huntington\_hill}, \texttt{divisor\_methods}, and
\texttt{paradoxes} modules), while the remaining tests cover the
prime-factor compositor, graph, nesting, spectral, and split strategies
that share the same crate.

\subsection{L0: Unit Tests}

L0 tests verify individual functions and edge cases in isolation.

\paragraph{Priority formula and allocation tests:}
\begin{itemize}
  \item Equal-population two-state examples split evenly under all four divisor
    methods.
  \item A 3:1 population ratio with four seats returns the unambiguous 3--1
    allocation under all four divisor methods.
  \item Huntington-Hill single-state, constitutional-floor, total-seat, and
    fewer-seats-than-states panic behavior are covered.
\end{itemize}

\paragraph{Comparison and bias tests:}
\begin{itemize}
  \item Jefferson is checked against Huntington-Hill on a synthetic large-state
    bias fixture.
  \item Adams is checked against Huntington-Hill on a synthetic small-state bias
    fixture.
  \item All four divisor methods are checked for requested house-size totals and
    minimum one-seat-per-state output.
\end{itemize}

\paragraph{Paradox checks:}
\begin{itemize}
  \item \texttt{check\_alabama\_paradox} returns no events for Huntington-Hill,
    Webster, Adams, and Jefferson over a synthetic 5-state population vector and
    house sizes 5 through 50.
  \item Empty and single-house-size inputs return no events.
  \item A monotone non-decrease property test checks all four divisor methods
    over house sizes 3 through 30 on a three-state fixture.
\end{itemize}

\subsection{L1: Integration Tests}

L1 tests verify complete apportionments on synthetic examples with known correct
answers.

\paragraph{Synthetic divisor examples:}
\begin{itemize}
  \item Equal-population and 3:1 ratio fixtures exercise all four divisor
    methods through the public \texttt{apportionment\_divisor()} API.
  \item Bias fixtures compare Adams and Jefferson against Huntington-Hill.
\end{itemize}

\subsection{L2: Real-Data Regression Tests}

L2 Census Bureau official-apportionment fixtures are not yet checked in for this
crate.  They are the next evidence step for J.6: materialize 2000, 2010, and
2020 population/seat tables, compare canonical output strings, and add hash
records for publication provenance.
