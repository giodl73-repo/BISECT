\section{Evaluation Plan}

Current evidence is L0 behavior for improvements, no-ops, rejected moves, and
staged methods; L1 CLI and RPLAN sidecar tests; and a public U.18 parent/child
lineage package in the package corpus. A benchmark-tier 100-unit synthetic grid
package now records parent-plan lineage and no-improvement behavior over a
larger audited input. Empirical distributions of accepted moves, objective
deltas, and tabu/LNS comparisons are future evidence.

The representative CLI shape is:
\begin{verbatim}
bisect improve --method one-move \
  --plan input.rplan --context input.rctx \
  --out-dir out --tolerance 34
\end{verbatim}
The grid10 benchmark package records the equally important no-improvement case:
the search ran, preserved the parent assignment, and still emitted auditable
summary and lineage artifacts.
