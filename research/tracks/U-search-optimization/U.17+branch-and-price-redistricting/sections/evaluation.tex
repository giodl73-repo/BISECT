\section{Evaluation Plan}

The near-term evidence is L0 pricing/master tests, an L1 exact package test,
and a public U.17 golden package in the package corpus. Comparisons with
branch-and-cut, formulation-only reports, and heuristic construction are future
work until command transcripts and larger fixture outputs are attached.

The CLI shape for the exact fixture path is:
\begin{verbatim}
bisect exact --method branch-and-price \
  --context fixture.rctx --out-dir exact \
  --districts 2 --tolerance 1.0
\end{verbatim}
Passing \texttt{--formulation-only} changes the claim: the output can document
the generated master but must not be described as a solved exact plan.
