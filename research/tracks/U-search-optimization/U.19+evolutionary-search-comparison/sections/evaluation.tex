\section{Evaluation Plan}

Current evidence includes L0/L1 crossover, mutation, determinism, selected
package verification tests, and a public U.19 selected-frontier package in the
package corpus. Seed/objective sensitivity, VRA-aware objective disclosure, and
frontier-quality comparisons remain future empirical work.

The representative CLI shape is:
\begin{verbatim}
bisect pareto --state TT --year 2020 \
  --selected-frontier-index 1 \
  --selected-frontier-context context.rctx \
  --selected-frontier-out selected-frontier
\end{verbatim}
Changing the base seed, objectives, population size, or generation budget can
change the frontier and therefore the meaning of index 1. Those parameters are
part of the evidence, not incidental run decoration.
