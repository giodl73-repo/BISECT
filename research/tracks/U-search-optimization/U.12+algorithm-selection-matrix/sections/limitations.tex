\section{Limitations}

The matrix is a decision aid, not a proof that any chosen workflow is legally
sufficient or politically fair. It must be revised when new algorithm families,
CLI surfaces, or audit profiles are added.

The current draft is framework-level evidence. Before publication-ready status,
each row should cite a concrete command transcript, fixture, benchmark, or
paper-specific evidence placeholder. Rows for exact optimization and LNS should
remain staged until U.16 through U.18 document their implementation surfaces.
