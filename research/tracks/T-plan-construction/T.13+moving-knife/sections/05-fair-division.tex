\section{The Fair-Division Argument}
\label{sec:fair-division}

\subsection{What the Argument Can Say}

The fair-division value of \mka{} is procedural intelligibility. A commission
can explain the structure step in ordinary geometric language:

\begin{quote}
We rotated a public set of candidate sweep directions across the current
region. For each direction, the split point was fixed by population balance.
We scored both halves by the same compactness rule and chose the direction
whose weaker half scored best.
\end{quote}

That statement is accurate for the implemented node-level algorithm. It is also
auditable: an outside reviewer can replay the projection order and cumulative
population table for each angle.

\subsection{What the Argument Cannot Say Yet}

\mka{} should not be described as a complete legal certificate. Several
boundaries matter:

\begin{itemize}
\item it optimizes a local bisection score, not the global Reock score of all
  final districts;
\item current scoring uses centroid points rather than full polygon boundary
  vertices;
\item the local rebalance can alter the selected sweep assignment;
\item VRA, community, nesting, county, and partisan fairness criteria are
  outside the split objective;
\item final-plan claims require archived assignments and independent scoring.
\end{itemize}

These limits do not weaken \mka{} as a construction primitive. They make its
publication boundary clear.

\subsection{Court-Facing Language}

The strongest court-facing formulation is modest:

\begin{quote}
The Moving-Knife run used a predetermined, geography-only sweep procedure to
generate each recursive split. The selected split at each node maximized the
minimum of the two implemented Reock-approximation scores among the tested
orientations. The run package records enough information for an independent
reviewer to reproduce the selected orientations and verify the final plan
against statutory criteria.
\end{quote}

Once per-node angle manifests and polygon-boundary Reock are implemented, this
language can be strengthened. Until then, the paper should emphasize auditability
and algorithmic neutrality, not legal sufficiency.

\subsection{Relationship to METIS and CVD}

\metis{} can be explained as minimizing graph boundary. \cvdgeo{} can be
explained as assigning tracts to geographic centers. \mka{} can be explained as
rotating a ruler across the map and choosing the split that best avoids a weak,
elongated half. The last explanation is unusually visual, which is why this
paper is useful even before the certification package is complete.
