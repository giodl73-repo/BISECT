\begin{abstract}
\noindent
Centroidal Voronoi Districts (\cvd{}, T.10) now has two implemented metric
modes inside BISECT: graph-distance CVD, which assigns tracts by BFS hop count,
and geographic CVD, which assigns tracts by Euclidean distance after projecting
tract centroids with the platform's spherical Albers transform. This companion
paper explains why the geographic mode exists, how it is wired into recursive
bisection, and what still separates the implementation from a publication-grade
state-by-state comparison.

The implemented changes over graph-distance CVD are concrete: a distinct
geographic seed domain; k-farthest initialization in projected coordinate
space; Lloyd-style coordinate updates snapped to the nearest tract; a
one-meter seed-displacement stopping rule; and an error path when geographic
CVD is requested without centroid data.
The current implementation still rebalances each local split toward one half
of node population, so ratio-aware geographic CVD splits for odd-seat nodes
remain an implementation and evidence gap.

The paper therefore treats NC/FL/WA tables as benchmark targets, not final
claims. Its main contribution is the audit contract: geographic CVD is now a
BISECT split primitive that can be verified from adjacency, population,
centroid, base-seed, and node-path inputs.
\end{abstract}
