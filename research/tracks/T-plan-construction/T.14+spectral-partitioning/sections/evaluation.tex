\section{Evaluation Plan}

The first evidence layer is fixture correctness: path graphs, two-clique graphs,
determinism, and recursive split handling. The current package layer adds a
method-produced six-unit spectral fixture and a 100-unit synthetic benchmark
package that exercise lineage, manifest hashes, and verifier acceptance. The
next evidence layer should compare spectral construction against METIS,
GeoSection, and clustering on selected real-data smoke cases.

The evaluation reading rule is therefore simple: fixture and package rows
support construction correctness and audit replay, while comparison rows are
research agenda items. The paper should not be read as evidence that spectral
construction beats METIS, GeoSection, capacity clustering, or regionalization
on plan quality; it creates the deterministic baseline needed to run that
comparison cleanly.
